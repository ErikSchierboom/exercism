# Word Count

Given a phrase, count the occurrences of each word in that phrase.

For example for the input `"olly olly in come free"`

```text
olly: 2
in: 1
come: 1
free: 1
```

## Setup

Go through the setup instructions for TypeScript to
install the necessary dependencies:
