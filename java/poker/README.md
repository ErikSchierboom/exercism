# Poker

Pick the best hand(s) from a list of poker hands.

See [wikipedia](https://en.wikipedia.org/wiki/List_of_poker_hands) for an
overview of poker hands.

# Running the tests

You can run all the tests for an exercise by entering

```sh
$ gradle test
```

in your terminal.
